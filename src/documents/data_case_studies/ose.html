``` isCaseStudy: true url: open-source-ecology name: Open Source Ecology type: case_studies location: Missouri, U.S.A description_short: Open Source Ecology is a network of farmers, engineers, architects and supporters, whose main goal is the manufacture of the Global Village Construction Set. description_long: Open Source Ecology (OSE) is a network of farmers, engineers, architects and supporters, whose main goal is the eventual manufacturing of the Global Village Construction Set (GVCS).

Focused on developing open source industrial machines that can be made for a fraction of commercial costs, with all designs shared freely online. The goal of Open Source Ecology is to create an open source economy - an efficient economy which increases innovation through open collaboration.

The current practical implementation of the GVCS is a life size LEGO set of powerful, self-replicating production tools for distributed production. The Set includes fabrication and automated machines that make other machines. Through the GVCS, OSE intends to build not individual machines - but machine construction systems, so that new machines can be built from existing machines, the GVCS is intended to be a kernel for building infrastructures of modern civilization. thumbnail: open_source_ecology.jpg video: date_established: 2003 people: Marcin Jakubowski category: infrastructure category_specific: link1_title: opensourceecology.org link1_url: http://opensourceecology.org/ link2_title: OSE Europe link2_url: http://oseeurope.org/ ```
